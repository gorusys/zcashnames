# Pricing (Protocol)

Claim prices are on-chain state set by signed `SETPRICE` memos. The wire format is in [Memo Format](/docs/protocol/memo-format#setprice). The current tiers are in the `status` RPC response — see [JSON-RPC Reference](/docs/indexer/rpc-reference).

## Activation

A `SETPRICE` takes effect at the block it lands in. All `CLAIM`s from that block forward use the new tiers. Existing registrations are unaffected.

## Bootstrap

The indexer ships with no hardcoded pricing. `status.pricing` is `null` until the first `SETPRICE` memo has been observed. `CLAIM`s before that point are silently rejected. On a fresh indexer, this resolves once the first historical `SETPRICE` replays from the chain.
