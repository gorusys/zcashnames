# Address Types

ZNS requires a **unified address** (`u1…` on mainnet, `utest1…` on testnet). No other address type is supported.

## Why unified only

ZNS memos travel inside Orchard shielded notes. Only unified addresses have an Orchard receiver. Sapling addresses (`zs1…`) and transparent addresses (`t1…`) do not — there is no shielded memo on them, so the indexer never sees the claim.

This is a deliberate choice to force adoption of Orchard. 

## If you don't have a unified address yet

You need a wallet that supports Orchard. See [Wallet Compatibility](/docs/use/wallets) for the current list.
