# What is ZcashNames?

A Zcash unified address looks like this:

```
u1gphl7vrklduuv96kpw4eetx4vrs8nnk7w9tuzvppyuuctw0tuskkpmfulrjapr05zh78p3chpxhx3tm28qau3uwd36k94vgucpxphyv5hkg36nhvr4ax
```

ZcashNames lets people pay you at `alice.zcash` instead. The name is short, memorable, and resolves to your unified address every time.


## Where it stands today

| Component | Status |
|-----------|--------|
| Testnet | Active. Registrations open, protocol stable. |
| Mainnet | Beta. Indexer running, treat as experimental software. |
| Wallets | Cake Wallet ships resolution (v5.8.0+). Other integrations in progress. |
| Spec | Canonical reference: [`openrpc.json`](https://github.com/zcashme/ZNS/blob/master/openrpc.json) plus the [Memo Format](/docs/protocol/memo-format) page. |

For a deeper picture of what's deployed and who's using it, see [Ecosystem](/docs/ecosystem).
